# 🚗 IronLot - Used Car Price Prediction & Automated Bidding Agent

## Overview
IronLot is a Machine Learning project that predicts the selling price of used cars at auction and uses those predictions to make intelligent bidding decisions.

## Features
- Data Cleaning
- EDA
- Feature Engineering
- LightGBM
- Optuna
- Automated Bidding Agent

## Technologies
Python, NumPy, Pandas, Scikit-learn, LightGBM, Optuna, Jupyter

## Files
- analysis_Harsh.ipynb
- train_Harsh.py
- agent_Harsh.py
- model_Harsh.pkl
- encoders_Harsh.pkl
- report_Harsh.pdf

## Results
- RMSE: 2042
- R²: 0.955
